## T14. `main.js`'de GPU Hızlandırma Aktifleştirildi

- **Seviye:** ORTA
- **Dosya:** `main.js` (çalışma kopyası, satır 3-6)
- **SORUN:** HEAD'de `process.env.ELECTRON_DISABLE_GPU = '1'` varken çalışma kopyasında bu satır **kaldırılmış**. GPU hızlandırma artık aktif.
- **RİSK:** Eğer uygulama GPU sorunları yaşayan bir ortamda çalışıyorsa (virtual machine, uzak masaüstü, eski GPU), `BrowserWindow` oluşturulamayabilir veya siyah ekran görülebilir. Bu satırın neden kaldırıldığı belli değil.
- **ÖNERİ:** GPU sorunu yoksa kalabilir, ama neden kaldırıldığı dokumente edilmeli. Geri getirmek gerekiyorsa `--disable-gpu` command line flag'i ile yapılabilir.

## EK: AI İÇİN GENEL NOTLAR

1. **Kod ES5 (`var`) ile yazılmış.** `let`/`const` kullanmıyor. Yeni kod eklerken mevcut stile uyum sağla.
2. **Tüm fonksiyonlar global.** Yeni fonksiyon eklerken isim çakışması olmamasına dikkat et.
3. **Firebase Compat API kullanılıyor.** `firebase.firestore.FieldValue.serverTimestamp()`, `firebase.auth.EmailAuthProvider.credential()` gibi compat API'ları kullan.
4. **`ls()` fonksiyonu** = localStorage getItem/setItem wrapper.
5. **`$()` fonksiyonu** = `document.getElementById()` alias.
6. **`uid()` fonksiyonu** = random ID üretici.
7. **`esc()` / `escJs()`** = HTML/JS escape fonksiyonları.
8. **Script yükleme sırası (index.html:275-288):** constants → store → utils → accounts → shortcuts → ui-core → e2e → messaging → firebase-data → settings → media → voice → calls → app-init
9. **Load order doğru:** `store.js` (2. sıra) tüm state'i tanımlar; `app-init.js` (15. sıra) `store.on()` ile abone olur.
10. **HEAD commit (26f21c6) vs çalışma kopyası:** HEAD tüm JS inline içerir. `src/` dosyaları `c1e0c8b` ve `4ac9fdc` commit'leri ile takibe alınmıştır.
11. **Değişiklik yapmadan önce** mevcut kodu yedekle veya git commit yap.
